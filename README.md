# Restaurant-Analysis-using-Python -
Data Analysis Internship Project

📌 Project Overview :

This repository contains the tasks and solutions completed as part of the Data Analysis Internship Program offered by Cognifyz Technologies.
The project focuses on analyzing restaurant datasets to extract meaningful insights using Python, Pandas, NumPy, Matplotlib, and Seaborn.

The internship tasks cover:

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Data Visualization
Statistical Analysis
Geographic & Review Analysis


🚀 Internship Details :

Organization: Cognifyz Technologies
Domain: Data Analysis
Internship Type: Virtual Internship
Tools Used: Python, Jupyter Notebook, Pandas, NumPy, Matplotlib, Seaborn
